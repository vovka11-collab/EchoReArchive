# EchoReArchive

EchoArchive is a modern open-source music client for Archive.org.

## Features

- 🎵 Stream music from Archive.org
- 🔍 Fast search
- 💿 Albums and artists
- ❤️ Favorites
- 📥 Offline caching
- 🌙 Dark theme
- ✨ Smooth animations
- 🎧 Background playback
- 📱 Built with Flutter

## Project Status

🚧 Under active development.
